# Evolve Kiosk

Standalone Next.js App Router kiosk application for trials, memberships, enquiries, and admin operations.

## Run

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Build

```bash
npm run build
npm run start
```

## Environment

Optional client-side PIN variables:

```bash
NEXT_PUBLIC_ADMIN_PIN=2580
NEXT_PUBLIC_STAFF_PIN=2580
```
